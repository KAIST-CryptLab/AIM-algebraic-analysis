# AIM-algebraic-analysis

This is an implementation of algebraic analysis on [AIM](https://eprint.iacr.org/2022/1387) one-way function and related primitives.

## Requirements
- [SageMath](https://www.sagemath.org/)
- [Magma](http://magma.maths.usyd.edu.au/magma/)

## Contents
We implement XL attack and groebner basis attack on AIM / [Rain](https://dl.acm.org/doi/abs/10.1145/3548606.3559353) / Even-Mansour cipher in toy parameters.

## How to use
```shell
python main.py
```
